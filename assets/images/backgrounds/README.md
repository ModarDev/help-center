# โฟลเดอร์สำหรับเก็บรูปพื้นหลัง

วางไฟล์รูปภาพสำหรับพื้นหลังในโฟลเดอร์นี้

## รูปแบบไฟล์ที่รองรับ:
- JPG / JPEG
- PNG  
- GIF
- WebP

## ขนาดที่แนะนำ:
- 1920x1080 หรือสูงกว่า
- อัตราส่วน 16:9 
- ไฟล์ไม่เกิน 5MB

## วิธีการเพิ่มรูปพื้นหลัง:

### วิธีที่ 1: ผ่านระบบ (แอดมินเท่านั้น)
1. ล็อกอินด้วยบัญชีแอดมิน
2. คลิกปุ่ม "จัดการพื้นหลัง" ที่มุมขวาบน
3. ลากไฟล์รูปภาพหรือคลิกเพื่อเลือกไฟล์
4. รูปจะถูกอัพโหลดและแสดงในระบบทันที

### วิธีที่ 2: วางไฟล์โดยตรง (FTP/File Manager)
1. อัพโหลดไฟล์รูปภาพลงในโฟลเดอร์นี้
2. รีเฟรชหน้าเว็บเพื่อดูรูปพื้นหลังใหม่

## การทำงานของระบบ:
- ถ้ามีรูปภาพมากกว่า 1 รูป จะมีการสไลด์อัตโนมัติทุก 5 วินาที
- สามารถควบคุมด้วยปุ่มลูกศรซ้าย-ขวา
- คลิกจุดตัวบ่งชี้ด้านล่างเพื่อไปยังรูปที่ต้องการ
- หยุดเวลาชั่วคราวเมื่อ hover เมาส์ที่หน้าต่างล็อกอิน

## การจัดลำดับ:
รูปภาพจะแสดงตามลำดับตัวอักษรของชื่อไฟล์
